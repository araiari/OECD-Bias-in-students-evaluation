Questa cartella contiene i modelli lineari bivariati, con come risposta il vettore [voti_matematica, voti_reading].

Sono stati costruiti 8 modelli sempre più grandi a partire dal risultato dell'elastic net.
I modello migliore è stato scelto tramite una cross-validation ed è risultato il modello 6.

-----------------------------------------------------
The folder contains the bivariate linear models and the scripts to compute them. 
The response is always the vector [math_evaluation, reading_evaluation].

We built 8 different models using more and more covariates to the result of the EN.
The best model (model 6) was chosen with a Cross Validation.
